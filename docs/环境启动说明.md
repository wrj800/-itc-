## 八、当前注意事项

1. Docker Prometheus 使用 `http://127.0.0.1:9090`，适合演示容器观测栈。
2. WSL Prometheus 使用 `http://<WSL-IP>:19090`，适合演示真实 Linux 节点监控；WSL IP 可能变化，启动脚本会自动打印。
3. 后端启动脚本会自动设置 `ICT_PROMETHEUS_WSL_BASE_URL`，把 Spring Boot 接到当前 WSL Prometheus。
4. `Blackbox Exporter` 当前配置了 `host.docker.internal:8080` 和 `host.docker.internal:5173`，后端/前端启动后会变成有效探测目标。
5. 本机已有其他 Docker 容器，`docker compose ps` 可能显示 orphan container 警告，不影响本项目服务。
